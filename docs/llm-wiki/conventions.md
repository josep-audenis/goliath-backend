# Conventions

## Documentation
- Put code structure in Graphifyy, not wiki.
- Put rationale, tradeoffs, constraints, and pitfalls in wiki.
- Prefer existing wiki page edits over new pages.
- Use `TBD` for unknown durable facts. Do not invent.

## Agent Behavior
- Read only relevant wiki pages.
- Link pages when one note depends on another.
- Keep entries short enough for scanning.

## Code
- Match existing project style.
- Add exact conventions here only when they are project-specific and durable.
